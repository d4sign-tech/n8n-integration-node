# @d4sign/n8n-nodes-d4sign

Community node for integrating n8n with the D4Sign API.

## Installation

Install in your n8n custom nodes environment:

```bash
npm install @d4sign/n8n-nodes-d4sign
```

## Credentials

Create a `D4Sign API` credential in n8n with:

- `tokenAPI`
- `cryptKey`
